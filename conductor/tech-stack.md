# Tech Stack - React Native Template

## Core Framework
- **React Native (0.85.0):** The core mobile framework, configured with the New Architecture enabled.
- **Expo (56.0.0):** Utilized for Expo modules and autolinking capabilities within a bare React Native project.

## Language
- **TypeScript (6.0.3):** For strict typing and enhanced developer experience.

## Infrastructure & Tooling
- **Metro:** The JavaScript bundler for React Native.
- **CocoaPods:** For managing iOS native dependencies.
- **Gradle:** For managing Android builds and dependencies.

## Testing
- **Jest (30.4.2):** For unit and integration testing.
- **React Test Renderer:** For component testing.

## Libraries
- **react-native-safe-area-context (5.8.0):** For handling safe area insets across different devices.
- **expo-modules-autolinking:** For automatic linking of Expo and other native modules.
- **React Navigation (7.3.3):** For seamless app navigation (Stack and Tab).
- **Zustand (5.0.14):** For lightweight and scalable global state management.
- **TanStack Query (5.101.0):** For efficient data fetching, caching, and synchronization.
